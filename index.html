Linux基础操作

2016-03-30

Linux中一切皆文件

管理员进入后提示符为”#”,普通用户为”$”;

快捷键

Tab    自动补齐

\         反斜杠强制换行

Ctrl

+u 清空至行首

+k 清空至行尾

+c 终止当前命令

基本命令

hostname    查看主机完整名称

ifconfig    查看以太网卡的ip地址

pwd(print working directory)    查看当前路径

sync    将缓存区的文件写入磁盘，避免关机或重启数据丢失

shutdown -h now (poweroff) (init 0)    关机

shutdown -r now (reboot) (init 6)    重启

shutdown -h +15 ‘host will be shutdown’    十五分钟后关机并留言

shutdown -c    取消关机

cd(change directory)    切换目录

date/hwcolock    日期

cal    日历

bc    计算器(scale=4保留4位小数)

uname    查看系统构架以及内核信息

-a 显示全部

-h 只显示内核信息

(cat /etc/issue 查看当前系统)

ls(List)    查看目录

-l 以长格式显示

-a 显示所有文件包括隐藏文件

-d 显示目录本身属性

-h 提供易懂容量单位

-F 在每一个文件的末尾加上一个字符说明该文件的类型。”@”表示符号链接、”|”表示FIFOS、”/“表示目录、”=”表示套接字。

-R 递归显示所有目录

文件操作

mkdir    创建文件夹

格式：mkdir [-p] 目录 ([递归]创建目录)

touch    新建空文件或更新文件时间戳

ln    创建文件连接（软连接）

格式：ln -s 源文件地址 连接文件路径

cp    复制

格式：cp [选项] 源文件地址 目标路径

-r 递归复制

-f 强制覆盖，不提示

-p 保持原文件属性不变

rm    删除

格式：rm [选项] 目标路径

-r 递归删除整个目录

-f 强制删除不提示

mv    移动

格式：mv [选项] 源文件地址 目标路径（若位置不变相当于改名）

cat    查看全部 （不足：大文件显示不方便）

格式：cat [选项] 文件地址

-n 在每行显示行号 (例：cat -n /etc/passwd 显示有多少个帐号)

more    大文件阅读（不足：向前翻页不便捷）

enter 下翻一行

空格 下翻一屏

q 退出

? 查看帮助

less    大文件阅读

Pgup向前翻页

Pgdu向后翻页

head/tail    截取文件首/尾部分

格式：head/tail -n 数字 文件地址 （默认10行）

结合使用实例：查看8到12行 head -n 12 file1.txt | tail -n 5

动态查看文件 tail -f 可用于查看报错等

wc    文本统计

格式：wc [选项] 文件地址

-l 计算文本总行数

-w 计算单词个数

-c 计算文件字节数

-m 计算文件字符数

grep 输出包含查找字符串的行

格式：grep [选项] ‘查找条件’ 目标文件

-i 忽略大小写

-v 反转查找，查找不符合条件

–color 突出查找条件

查询条件双引号中指定条件

^…… 表示以……开始

……$ 表示以……结束

^$ 表示空行

zip    压缩处理

格式：zip 压缩包名称 被压缩文件 （结合-r可打包目录）

unzip    解压缩

格式：unzip -d 目标文件夹 压缩文件.zip

gzip/bzip2    linux压缩

格式：gzip/bzip -9 文件名    压缩

gzip/bzip -d 文件名.gz/.bz格式    解压缩

tar    归档工具

tar 选项 归档文件 源文件    制作归档

tar 选项 归档文件 -C 目标目录    释放归档

-c 创建.tar格式归档文件

-x 解开.tar格式归档文件

-C 解包时指定释放的目标文件夹

-f 表示使用归档文件

-z 调用gzip执行压缩或者解压缩

-j 调用bzip2执行压缩或者解压缩

-v 输出详细信息

-t 列表查看包内文件

find    精确查找文件

格式：find 查找范围 查找条件

-name 按文件名称查找

-size 按文件大小查找

-mtime 按文件修改时间查找

-type 按文件类型查找(f 普通文件,d 目录,l 连接文件,b 块设备文件,c 字符设备文件)

实例：

-name可使用通配符 find /etc -name “resol*.conf”

-a 多条件同时满足 find /dev -type c -a -name “tty[1-3]”

-o 多条件满足任意一个

-size +/-大小 find /boot/ -size +2M

-mtime +/-天数 在n天之前/之内修改

补充命令

xargs

实例：ps -ef|grep memcache|cut -c 9-15|xargs kill -9 关闭memcache相关所有程序
